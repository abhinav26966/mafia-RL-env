# ───────────────────────────────────────────────────────────────────────────
# Training stack for DECEIT Werewolf GRPO post-training.
#
# Two compute targets:
#   1) Colab Free  — Tesla T4 (16 GB).    Use Gemma-3 1B (debug + initial run).
#   2) Colab Pro / HF compute — A100 40GB. Use Qwen3-1.7B (final run).
#
# DO NOT pin torch/CUDA wheels here — Colab provides them. Installing this
# file via `pip install -r requirements-train.txt` should add packages on top
# of Colab's pre-installed torch.
#
# Only the THREE critical pins below are hard. Everything else is loose so
# pip can resolve mergekit, accelerate, datasets, etc. without conflict.
# Past pinning experiments showed:
#   - transformers 5.x breaks Unsloth + TRL 0.20 (Trainer source-edit regex)
#   - pydantic 2.12 breaks mergekit (raw torch.Tensor in pydantic schemas)
#   - trl 0.21+ may or may not match — 0.20 is what we tested with
# ───────────────────────────────────────────────────────────────────────────

# Hard pins — needed for known-stable behavior
transformers==4.46.3
trl==0.20.0
pydantic==2.10.6

# Soft requirements — let pip resolve

mergekit
huggingface-hub

# Tracking / plotting (also soft)

matplotlib

seaborn
