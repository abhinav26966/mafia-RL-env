# Training stack for DECEIT Werewolf GRPO post-training.
#
# DESIGN DECISION (2026-04-26): we do NOT pin transformers, trl, or
# accelerate here. Reason:
#   - Unsloth's pyproject pins transformers to >4.51.3, <=4.57.6
#   - Pinning transformers manually below that range causes pip's resolver
#     to fail on `pip install -r requirements-train.txt`
#   - The notebook installs `trl --no-deps` so trl doesn't drag its own
#     transformers in either
# Net effect: Unsloth picks the transformers version, everything resolves.
#
# Only HARD PIN: pydantic 2.10.6 (mergekit's torch.Tensor schemas break on
# pydantic 2.12, and 2.11 has internal API changes that bite us).
# ───────────────────────────────────────────────────────────────────────────

# Critical pin — only one
pydantic==2.10.6

# Soft requirements — let pip resolve
datasets
accelerate
peft
trackio
matplotlib
pandas
seaborn
huggingface-hub
